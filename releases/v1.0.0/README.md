## 🎯 主要功能

### 1. 路由表可视化
- 📡 **活动路由区域**：显示临时路由（系统重启后丢失）
- 💾 **持久路由区域**：显示永久路由（系统重启后保留）
- 🔄 **协议切换**：支持IPv4和IPv6协议切换
- 📊 **实时统计**：显示各类路由数量

### 2. 路由管理
- ➕ **添加路由**：支持指定目标网络、网关、接口等参数
- ➖ **删除路由**：安全删除不需要的路由
- 🔧 **持久路由**：勾选选项使路由永久保存
- 🌐 **接口选择**：自动检测系统网络接口，支持指定路由出口

### 3. 智能权限管理
- 🔑 **权限检测**：自动检测当前管理员权限状态
- 🚀 **自动提升**：需要时自动请求UAC权限
- 📝 **状态显示**：实时显示权限状态和连接状态

## 💡 使用说明

### 启动程序
1. **双击 `RouteManager.exe`**
2. **权限提示**：在UAC对话框中点击"是"
3. **开始使用**：程序启动后显示完整的路由管理界面

### 查看路由
1. **选择协议**：使用顶部的IPv4/IPv6单选按钮
2. **浏览路由**：在对应区域查看活动路由和持久路由
3. **查看状态**：底部状态栏显示详细信息

### 添加路由
1. **点击"添加路由"按钮**
2. **填写信息**：
   - 目标网络：如 `192.168.1.0`
   - 子网掩码：如 `255.255.255.0`
   - 网关：如 `192.168.1.1` 或 `On-link`
   - 接口：从下拉框选择或"自动选择"
3. **可选设置**：
   - 勾选"持久路由"使路由永久保存
   - 设置跃点数（可选）
4. **确认添加**：点击确定完成路由添加

### 删除路由
1. **选择路由**：在表格中点击要删除的路由
2. **确认删除**：点击"删除路由"按钮
3. **自动完成**：程序自动执行删除操作

## 🛠️ 故障排除

### 权限问题
**问题**：程序启动时提示权限不足
**解决方案**：
- 确保使用管理员账户登录
- 在UAC提示中点击"是"
- 或右键"以管理员身份运行"

### 启动问题
**问题**：程序无法启动
**解决方案**：
- 检查系统完整性
- 重新下载程序文件
- 确保Windows版本兼容

### 路由操作失败
**问题**：添加/删除路由失败
**解决方案**：

- 按照程序建议操作
