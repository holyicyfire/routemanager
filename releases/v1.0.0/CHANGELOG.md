# 更新日志

## v1.0.0 - 2024年11月12日

### 🎉 首次发布

#### ✨ 核心功能
- **完整路由管理**：支持添加、删除、查看系统路由
- **双协议支持**：同时支持IPv4和IPv6路由管理
- **分离显示**：活动路由和持久路由独立显示
- **智能权限**：自动检测和提升管理员权限

#### 🔧 技术特性
- **独立运行**：无需Python环境，单文件部署
- **图形界面**：纯Tkinter GUI，无控制台窗口
- **网络接口**：自动检测系统接口，支持指定路由出口
- **持久路由**：支持添加系统重启后保留的路由

#### 🛠️ 开发工具
- **完整打包**：PyInstaller一键打包工具
- **多脚本支持**：bat和ps1多种打包选项
- **详细文档**：完整的用户和开发者文档
- **版本管理**：Git仓库和版本控制

#### 📋 用户体验
- **直观界面**：双区域清晰显示路由信息
- **实时状态**：状态栏显示权限和统计信息
- **错误处理**：详细错误提示和解决方案
- **操作引导**：完整的使用说明和指导

## 贡献指南

### 开发环境
1. 克隆仓库：`git clone https://github.com/holyicyfire/routemanager.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python route_manager.py`
4. 打包测试：运行打包脚本

### 提交代码
1. Fork仓库
2. 创建功能分支
3. 提交更改
4. 创建Pull Request

## 版本说明

### 版本号规则
- **主版本号**：重大功能更新，可能包含破坏性变更
- **次版本号**：新功能添加，向后兼容
- **修订版本号**：Bug修复和改进

### 发布周期
- **主版本**：3-6个月更新一次
- **次版本**：1-2个月更新一次
- **修订版本**：按需更新（Bug修复）

### 获取更新
- **GitHub Releases**：https://github.com/holyicyfire/routemanager/releases
- **自动更新**：程序可支持在线更新检查（计划功能）
